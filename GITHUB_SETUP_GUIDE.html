Hosting FitJain on GitHub Pages

Free hosting · Works on Android · Installable as an app · Takes ~10 minutes

1 Create a free GitHub account

Go to github.com and sign up. Pick any username. Verify your email.

2 Create a new repository

After logging in, click the green "New" button (top left) or go to github.com/new.

  • • Repository name: fitjain
  • • Set to Public (required for free GitHub Pages)
  • • Leave everything else as default
  • • Click "Create repository"

3 Upload your files

On the new repository page, click "uploading an existing file" link.

Drag and drop ALL these files from the ZIP you downloaded:

fitjain/ ├── index.html ← main app file ├── manifest.json ← makes it installable ├── sw.js ← makes it work offline ├── icon-192.png ← app icon └── icon-512.png ← app icon (large)

Scroll down, write commit message: "Initial upload" — then click "Commit changes".

4 Enable GitHub Pages

In your repository, go to: Settings → Pages (left sidebar)

  • • Source: Deploy from a branch
  • • Branch: main — folder: / (root)
  • • Click Save

Wait 1–2 minutes. Refresh the page. You'll see:

✓ Your site is live at: https://YOUR-GITHUB-USERNAME.github.io/fitjain/
Example: If your GitHub username is "maharsh2024", your app will be at:
https://maharsh2024.github.io/fitjain/

5 Install on your Android phone

Open the URL above in Chrome on your Android phone and:

  • Open Chrome → go to your GitHub Pages URL
  • Tap the 3-dot menu (⋮) in Chrome's top right
  • Tap "Add to Home screen" or "Install app"
  • Tap Add / Install to confirm
  • FitJain now appears on your home screen like a real app!
  • Open it — it works offline too (no internet needed after first load)
✓ What you get on Android: Full-screen app (no browser bar) · Works offline · Home screen icon · Fast loading · All data saved on your phone locally

Updating the app in future

When you want to make changes (e.g. add a new meal or update your goal weight):

# Option A — GitHub website (easiest) Go to github.com/YOUR-USERNAME/fitjain Click index.html → pencil icon (Edit) → make changes → Commit # Option B — Replace the file Go to repository → Add file → Upload files Upload new index.html → Commit (it replaces the old one) Changes go live in ~1 minute
